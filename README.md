# CheckerZ 🏁

A modern, visually appealing Checkers game built with **JavaScript** and bundled as a desktop application using **Electron**.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mesofot23rd/checkerz.git
   ```
2. Navigate to the project directory:
   ```bash
   cd checkerz
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Game
To start the application in development mode:
```bash
npm start
```

## ⌨️ Controls
- **Mouse:** Click on a piece to select it, then click on a highlighted square to move.
- **Ctrl + L:** Restart the game instantly.
- **F11:** Toggle Fullscreen mode.
